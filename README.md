# INOXA

Smart Home Energy Management Dashboard

##  Overview

INOXA is a modern smart home energy management system designed to monitor, analyze, and optimize household electricity consumption in real time.

The dashboard provides live energy tracking, device monitoring, automation scheduling, analytics visualization, eco scoring, and AI-based energy recommendations through an interactive and responsive user interface.

---

##  Features

- Real-time power monitoring
- Energy analytics dashboard
- Smart device control system
- Dark / Light mode support
- AI energy assistant
- Smart alerts and notifications
- Live charts using Chart.js
- Room-wise energy breakdown
- Automation schedules
- Fully responsive UI
